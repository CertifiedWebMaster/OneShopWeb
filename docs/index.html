Storefront style guide

This is a guide for making componets that can be used on any E-Com Plus store without breaking it's visual identiy.
You can and should use Bootstrap Utilities, they will follow the current installed theme variables.



Colors

The store owner set two brand colors that together with the theme colors, will fill all the store elements.
To use these colors, take a look in bootstrap Colors Utilities


Primary

The primary is the action color, and should be used to drive attention to the main tasks that shoud be done while using the app. It is meant to be used in major interactive elements of the page.

Use examples: Call to actions Obrigatory forms Check boxes Radio buttons Toggles




Secondary

The secondary is the main brand color, used to spotlight elements with the store brand identity. It should be used to drive attention in the elements with relevant info, but where the user is not intended to take action

Use examples: Badges Sales tags Hotmarks

Notifications colors

Success, info, warning and danger are the notificantions colors. These colors are used to report something, facilitating user understanding of the information. This colors usage should be avoided on elements that don't give notificantions or any feedback.

Use examples: Alerts Badges Confirmations Feedback messages




Neutral colors

Light, white, dark and transparent are the colors that will fill most of the content. It should be mainly used in pages and major elements backgrounds.

Use examples: Backgrounds Modals Cards Borders Separators

Colors

The .block_color classes will set the background color as the chosen one. The class also set text color to black or white, depending on the best to contrast with the background.
