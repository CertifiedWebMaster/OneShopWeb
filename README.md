# storefront-template

PWA &amp; JAMstack e-commerce theme for E-Com Plus stores

## Template reference

### Element identifiers

IDs and classes to identify elements that you may
want to add custom functionalities or widgets:

- `#menu`
- `#main`
  - `#header`
    - `#logo`
    - `#user-button`
    - `#cart-button`
    - `#search-form`
  - `#content`
  - `#overlay`
