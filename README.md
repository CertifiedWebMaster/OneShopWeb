# storefront-template

PWA &amp; JAMstack e-commerce theme for E-Com Plus stores

## Template reference

### Element identifiers

We're using IDs and classes to identify elements that you may
want to add custom functionalities or widgets, those
elements are listed below:

- `#menu`
- `#main`
  - `#header`
    - `#navbar`
      - `#logo`
      - `#user-button`
      - `#cart-button`
      - `#search-form`
  - `#content`
  - `#overlay`
